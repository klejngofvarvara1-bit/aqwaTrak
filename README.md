Приложение Aquatrac предназначено для отслеживания ежедневного потребления воды и формирования полезной привычки регулярного питья. Основную проблему, которую решает приложение, можно описать как отсутствие контроля и регулярности в потреблении воды.Приложение создано для студентов и школьников, людей с активным образом жизни
